Need Support?
=============

There are several ways to get support for the CKEditor Pack for TYPO3.

Community Support
-----------------

If you run into any issues or have new ideas, please open an issue or feature request in our GitHub repository:
`Github Issues <https://github.com/T3Planet/rte_ckeditor_pack/issues>`_.
To help us resolve things faster, kindly describe your concern in detail and use the predefined issue templates when creating your ticket.

.. rst-class:: horizbuttons-attention-m
   - `Report Issues <https://github.com/T3Planet/rte_ckeditor_pack/issues>`_

Slack Community
---------------

Join the `TYPO3 Slack Workspace <https://typo3.slack.com/archives/C09UKJY15DF>`_ to connect with other TYPO3 users and experts.  
Use the dedicated channel `#ckeditor-pack <https://typo3.slack.com/archives/C09UKJY15DF>`_ for discussions, questions, and technical help.  
If you are not registered yet, please follow the official TYPO3 Slack onboarding guide.

.. rst-class:: horizbuttons-attention-m
   - `Join TYPO3 Slack <https://typo3.slack.com/archives/C09UKJY15DF>`_

Personal or Professional Support
--------------------------------

If you require private, in-depth, or priority assistance, you may contact the developers or the `T3Planet support <https://t3planet.de/support>`_ team directly.

.. rst-class:: horizbuttons-attention-m

   - `Submit Ticket <https://t3planet.de/support>`_
